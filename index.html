Kafka4m

Kafka4m (kafka for monix) is a bit like the fs2kafka, but a bit more opinionated, in that it also brings in a typesafe configuration and application endpoint.

It can be used as an ETL job for getting data into or out of Kafka using a Kafka Stream.
